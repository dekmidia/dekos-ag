# Playbook: Producao de Criativos e Artes no Canva

## Principio Fundamental

Design sem estrategia e decoracao.
Cada peca deve comunicar um objetivo claro em menos de 3 segundos.
O squad Creative nao executa templates — pensa, direciona e entrega criativos de agencia.

---

## Fase 1 — DECODE: Decodificando o Briefing

Se o briefing for vago, ASSUME o melhor cenario e apresenta junto com a direcao criativa.

| Elemento      | O que Capturar                              | Se Nao Informado                        |
|---------------|---------------------------------------------|-----------------------------------------|

## Fase 2 — DIRECTION: Direcao Criativa

### Estrutura da Direcao Criativa

Todo briefing gera uma Direcao Criativa com:
- **Conceito**: nome + logica + emocao alvo (ex: "Transformacao Comprovada")
- **Hierarquia Visual**: o que o olho ve 1o, 2o e 3o
- **Paleta**: 3 cores max com HEX
- **Tipografia**: heading font + body font
- **Formato e dimensao**

### Magic Studio — Uso Estrategico

| Ferramenta         | Melhor Uso                                       | Prompt/Dica                                          |
|--------------------|--------------------------------------------------|------------------------------------------------------|
| Magic Design       | Gerar 8-10 opcoes de layout com 1 prompt         | [tipo] + [objetivo] + [estetica] + [plataforma] + [nicho] |
| Magic Write        | Gerar 5 opcoes de headline para escolher         | Ative Brand Voice com exemplos da marca              |
| Magic Switch       | Adaptar 1 design para multiplos formatos         | Crie no 1080x1080, switch para Stories e demais      |
| Dream Lab          | Gerar imagens AI unicas para backgrounds         | Adicione: "commercial photography, 8k quality"       |
| Magic Animate      | Transformar posts estaticos em animados          | Animacoes sutis — fade ou slide, nunca todas juntas  |
| Background Remover | Isolar produtos e pessoas de fotos               | Funciona melhor com alto contraste sujeito/fundo     |

### Prompt Padrao para Magic Design

```
[Tipo de peca] + [Objetivo] + [Tom/Estetica] + [Plataforma] + [Nicho]

Exemplo:
"Instagram post for product launch, bold and energetic aesthetic,
Gen Z audience, fitness supplement brand, strong CTA visible"
```

### Prompt Padrao para Dream Lab (Imagens AI)

```
[Sujeito detalhado] + [Ambiente] + [Estilo visual] + [Iluminacao] + [Qualidade]

Exemplo:
"Confident Brazilian woman smiling at camera, modern minimalist office,
soft natural light from window, commercial photography style, 8k quality"
```

### Tecnicas Avancadas de Design

**Hierarquia Visual em 3 Camadas**
1. ANCHOR: elemento maior — capta atencao em 0.5s (headline gigante ou imagem hero)
2. SUPPORT: texto secundario — beneficio, prova social ou contexto
3. ACTION: CTA claro — botao, tag ou seta direcional

**Regra do Espaco Negativo**
- 40% do canvas deve respirar (espaco estrategico vazio)
- Nunca preencher todos os cantos — espaco vazio comunica premium
- Espaco minimo entre elementos: 24px

**Tecnica de Profundidade**
- Camada fundo: cor solida, gradiente sutil ou textura leve
- Camada intermediaria: imagem ou forma geometrica
- Camada frente: texto principal e CTA (sempre o mais legivel)

## Fase 4 — COPY: Copy Estrategico Integrado

### Framework para Anuncios (Meta e Google Ads)

```
HEADLINE:     [Beneficio ou Dor Transformada]    — max 40 caracteres
SUBHEADLINE:  [Como funciona ou Prova Social]    — max 80 caracteres
CTA:          [Verbo de acao + Beneficio]         — max 20 caracteres
```

### CTAs de Alta Conversao

- "Quero meu resultado agora"
- "Acessar gratuitamente hoje"
- "Ver como funciona"
- "Comecar minha transformacao"
- "Quero essa oportunidade"

### Framework para Posts Organicos

```
HOOK VISUAL:  Pergunta, numero impactante ou afirmacao ousada
VALOR:        Informacao ou transformacao prometida
DIRECIONAL:   Instrucao de proximo passo (Salve, Comente, Link na bio)
```

### Estrutura de Carrossel Perfeito

| Slide      | Conteudo                                               |
|------------|--------------------------------------------------------|
| 1 (Capa)   | Hook irresistivel + promessa clara                     |
| 2 a N      | 1 ponto de valor por slide — titulo + max 3 linhas     |
| Penultimo  | Resumo ou sintese dos pontos                           |
| Final      | CTA forte + instrucao clara + @perfil ou logo          |

## Fase 5 — QA: Checklist de Qualidade

### Visual

- [ ] Hierarquia clara — headline domina visualmente
- [ ] Contraste adequado — texto legivel sobre o fundo (minimo 4.5:1)
- [ ] Canvas respira — nao esta sobrecarregado de elementos
- [ ] Paleta consistente — maximo 3 cores principais
- [ ] Tipografia — maximo 2 familias de fontes
- [ ] Elementos alinhados — usar guias automaticas do Canva
- [ ] Logo presente — discreto mas visivel (60-100px, canto inferior)

### Tecnico

- [ ] Dimensao correta para o formato
- [ ] Zona segura respeitada (Stories: 250px topo e base livres)
- [ ] Texto nao ultrapassa 20% da area (apenas para Meta Ads)
- [ ] Export PNG 2x para estaticos
- [ ] Export MP4 para animados (max 15s para Stories)
- [ ] Export PDF com sangria 3mm para impressos

### Estrategico

- [ ] Objetivo da peca esta claro em 3 segundos
- [ ] CTA presente, visivel e com contraste maximo
- [ ] Identidade de marca reconhecivel sem precisar ler o texto
- [ ] Preview testado no celular antes de publicar
