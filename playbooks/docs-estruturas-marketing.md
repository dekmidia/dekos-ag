# Playbook: Estruturas de Marketing e Raciocínio Estratégico

Este playbook contém metodologias avançadas de raciocínio, desconstrução de problemas e formulação de estratégias (SEO, campanhas, conteúdo) extraídas de práticas profissionais de Marketing Digital. **Utilize essas estruturas ao invés de criar saídas genéricas ou intuição pura.**

## 1. O Método DECAF (Análise Abrangente)

Ideal para destrinchar Tópicos Complexos, como auditoria de SEO ou pesquisa de palavras-chave.

- **Decompose (Decompor):** Quebre o grande tópico nos seus subtópicos fundamentais.
- **Elicit (Elicitar):** Explique cada subtópico de forma clara e aprofundada.
- **Connect (Conectar):** Demonstre como as peças (subtópicos) se relacionam dentro da estratégia.
- **Arrive at the Solution (Solucionar):** Indique quais aspectos importam mais para diferentes cenários ou ICPs (Ideal Customer Profiles).
- **Finalize the Goal (Finalizar):** Gere um resumo executivo focado em insights acionáveis.

## 2. Abordagem de Sistemas (Systems Thinking)

Ideal para analisar canais e gargalos no funil de Marketing.

- **Pilares Extrovertidos:** Comunicação, Pessoas, Métricas (Measurements), Resultados (Outcomes) e Redes (Networks).
- **Modelo Trinário Introvertido:** Estrutura, Clareza e Propósito.
- *Aplicação:* Mapeie como as campanhas se interligam nesses sistemas para identificar o maior ponto de alavancagem.

## 3. Playbook de Solução de Problemas (Problem Solving Framework)

Ideal para otimização de On-Page, CRO e análise de performance de Ads.

1. **Análise de Componentes (Component Analysis):** Liste todos os elementos.
2. **Mapeamento de Relacionamentos (Relationship Mapping):** Avalie as dependências.
3. **Identificação de Problemas (Problem Identification):** Destaque os desafios específicos encontrados em cada componente.
4. **Desenvolvimento de Soluções (Solution Development):** Crie hipóteses para cada problema.
5. **Plano de Implementação (Implementation Planning):** Defina passos acionáveis, milestones e métricas de sucesso.

## 4. O Framework Taxonômico

Ideal para criar Categorização de Conteúdos, Hubs de Artigos ou Arvores de Palavras-Chave.

- Liste Categorias Amplas (Broad Categories).
- Quebre em Subcategorias de Cauda Longa.
- Defina o Fluxo Lógico partindo do conceito mais genérico no topo para os dados mais específicos na ponta.

## 5. Estrutura de Propostas: Conexão de Pontos (High-Ticket)

Para qualquer serviço (Ads, SEO, Social Media), a proposta deve seguir este fluxo lógico e persuasivo:

1.  **A Jornada (Contexto/Storytelling):** Inicie com a dor ou o sonho do cliente final. Gere identificação imediata.
2.  **A Oportunidade/FOMO (O Custo da Inação):** Apresente dados (volumes de busca, mercado) que mostram o que o cliente perde ao não agir agora.
3.  **O Mecanismo (A Solução):** Apresente o serviço como a ponte óbvia. Use metáforas (ex: "Recepcionista 24h").
4.  **Transparência (Investimento):** Separe claramente o valor de Gestão (Fee) dos custos de terceiros/plataformas.
5.  **O Próximo Nível (Autoridade):** Use portfólios ou casos de sucesso para pintar o futuro ideal.
6.  **Call to Action (CTA):** Próximos passos simples e diretos para o fechamento.

---
> **Lembrete para o Agente:** A qualidade de uma copy ou de um plano de marketing não está nas "palavras bonitas", mas na força irrefutável da estrutura usada. Escolha o framework que responde melhor à necessidade atual do usuário e guie a sua resposta nos passos descritos acima.
