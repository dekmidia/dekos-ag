# Checklist: Diagnostico Digital

Execute antes de apresentar qualquer diagnostico para aprovacao humana.

## Dados e Veracidade
- [ ] PageSpeed score mobile verificado (nao estimado)
- [ ] Posicao no Google verificada para "[segmento] em [cidade]"
- [ ] Status de anuncios verificado (Meta Ad Library)
- [ ] Data do ultimo post no Google Meu Negocio verificada
- [ ] Zero numeros inventados

## Conteudo
- [ ] Pelo menos 1 ponto positivo antes dos problemas
- [ ] Nota por canal para todos os 5 canais
- [ ] Top 3 oportunidades com timeline (7d / 30d / 90d)
- [ ] Impacto estimado em leads/mes presente

## Limites Inegociaveis
- [ ] Nenhum preco ou pacote DekMidia no corpo
- [ ] Tom consultivo, nao comercial

## Entrega
- [ ] CTA para conversa de 20 minutos presente
- [ ] Link Calendly incluido (ou placeholder)
- [ ] Arquivo salvo em `PROJETOS/_prospeccao/diagnosticos/[EMPRESA]-[DATA].md`
- [ ] Mensagem de envio gerada e pronta
