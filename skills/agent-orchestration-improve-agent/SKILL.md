# Agent Performance Optimization

Skill acionada quando o sistema precisa auto-auditar seu comportamento ou gerar instruções ainda melhores.

## Estratégias de Melhoria

1. **Identifique a Falha (Baseline):** Descubra exatamente em que momento o agente costuma alucinar ou ignorar a instrução (ex: criar pastas onde não devia).
2. **Implemente Chain-of-Thought (CoT):** Obrigue o agente a pensar em blocos antes de gerar a resposta.
3. **Few-Shot Examples:** Insira exemplos positivos e negativos claríssimos de input e output esperado no prompt do agente.
4. **Valide:** Submeta as alterações.

Isso é a chave para a evolução contínua da inteligência do DEK-OS.
