# Landing Page Analyzer — Skill

## Descrição

Skill compartilhada de análise de landing pages. Verifica velocidade, presença de pixels de rastreamento, SEO on-page, Core Web Vitals e conformidade técnica.

## Categoria

⚙️ Engineering

## Usada por

- **Auditor** — Verificação de LP antes de publicar anúncios
- **QA Tester** — Testes completos antes de deploy
- **SEO Specialist** — Auditoria SEO on-page
- **Scout** — Avaliação de presença digital de prospects

## Verificações

- Tempo de carregamento (target: < 3s)
- Core Web Vitals (LCP, FID, CLS)
- Pixels instalados (Google Ads, Meta, TikTok)
- SEO on-page (title, meta description, h1, alt tags)
- HTTPS ativo
- Cookie consent e política de privacidade
- Links quebrados
- Responsividade mobile
