# Estratégia de Marketing Profissional

**Esta skill previne a geração de campanhas e textos genéricos.** Ela atua como um supervisor de qualidade que força a utilização de frameworks mentais rigorosos de marketing e SEO.

## O Que Fazer

Quando o usuário solicitar uma pesquisa de palavras-chave, um plano de marketing, uma estruturação de campanha, a otimização de uma página (On-Page SEO) ou textos persuasivos:

1. **PARE** a geração imediata de código ou texto!
2. Acesse o playbook corporativo: `playbooks/docs-estruturas-marketing.md` usando a ferramenta de visualização de arquivos do seu ambiente.
3. Escolha o framework listado lá que melhor resolve o problema do usuário (Ex: "Método DECAF" para explicar conceitos ou "Problem Solving Framework" para encontrar gargalos no funil de métricas).
4. Siga estritamente os passos do framework eleito.
5. Crie a saída do usuário espelhando essa estrutura com alta clareza.

Isso elimina "alucinações copy-paste" e traz alto rigor técnico para todas as suas entregas. O formato de saída será infinitamente valioso para o usuário.
