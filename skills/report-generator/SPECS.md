# Report Generator — Skill

## Descrição

Skill compartilhada de geração de relatórios em múltiplos formatos. Transforma dados brutos em relatórios legíveis para clientes via WhatsApp, PDF ou dashboards.

## Categoria

📋 Content

## Usada por

- **Reporting** — Relatórios diários e semanais
- **Data Scientist** — Relatórios de ROI e análise preditiva
- **Onboarding** — Relatórios de setup e progresso

## Formatos

- WhatsApp (texto curto com emojis)
- PDF/DekPanel (relatórios detalhados com gráficos)
- JSON (dados estruturados para dashboards)

## Templates

```
📊 Bom dia [Cliente]! Resumo de ontem:
💰 Investimento: R$ [valor]
👥 Leads gerados: [número]
💵 Custo por lead: R$ [valor]
📈 Tendência: [subindo/estável/caindo]
[Insight do dia]
```
